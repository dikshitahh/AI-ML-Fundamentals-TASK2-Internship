## Machine Learning Types

1. Supervised Learning
2. Unsupervised Learning

## DEFINITION
1. Supervised Learning:
It uses labeled data.

Examples:
- House Price Prediction
- Email Spam Detection

2. Unsupervised Learning
It uses unlabeled data.

Examples:
- Customer Segmentation
- Clustering

## ALGORITHM

1. Supervised Learning:

-> Linear Regression – It predicts continuous values such as house prices.
-> Logistic Regression – It is use for classification problems.
-> Decision Tree – It makes predictions using a tree-like structure.
-> Random Forest – It combines multiple decision trees for better accuracy.
-> K-Nearest Neighbors (KNN) – It classifies data based on nearby data points.

## Example
Predicting whether an email is spam or not.

2. Unsupervised Learning:

-> K-Means Clustering – It groups similar data points into clusters.
-> Hierarchical Clustering – It creates a hierarchy of clusters.
-> DBSCAN – It finds clusters based on data density.
-> Principal Component Analysis (PCA) – It reduces the number of features while preserving important information.

## Example
Grouping customers based on purchasing behavior.

## Conclusion

Supervised Learning uses labeled data to make predictions, while Unsupervised Learning discovers hidden patterns and relationships in unlabeled data.
