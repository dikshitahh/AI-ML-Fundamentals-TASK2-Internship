##Rule Based Program vs AI Program

----------------------------------------------------------------------------------
|          Rule Based Program                 |        AI program                |
----------------------------------------------------------------------------------
| It uses predefined rules written by humans. | It learns pattern from data.     |
| It cannot learn from experience.            | It improves through training.    |
| It works only for specific conditions.      | It can handle complex situations.|
|easy to understsand and debug.               | It may be harder to interpret.   |
----------------------------------------------------------------------------------

##Example
-> RULE BASED PROGRAM:
             if marks > 40:
                 print("Pass")
             else
                 print("Fail")

-> AI PROGRAM:
              Train model using previous student data
              Predict pass/fail automatically
