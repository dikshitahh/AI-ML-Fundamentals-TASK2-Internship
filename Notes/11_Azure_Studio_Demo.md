# Microsoft Azure AI Studio Demonstration

## Purpose

Microsoft Azure AI Studio is a cloud-based platform used to build, train, deploy, and manage AI and Machine Learning models.

## Demonstration Project: Loan Approval Prediction

In the demonstration, a Loan Approval Prediction model was developed using Azure AI Studio. The goal of the project was to predict whether a loan application would be approved based on applicant information and historical data.

## Key Steps Demonstrated

-> Data Upload and Preparation
-> Model Training
-> Model Evaluation
-> Prediction Generation

## Learning Outcome

Through this demonstration, I learned how Machine Learning models can be built and deployed using Azure AI Studio with minimal coding. I also understood how AI can be applied to solve real-world business problems such as loan approval prediction.
